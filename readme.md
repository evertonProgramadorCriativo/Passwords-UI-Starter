
Esse projeto foi desenvolvimento front-end focada em **Vue 3**,


##  Scripts 

No diretório do projeto, você pode rodar:

###  Desenvolvimento
```bash
npm run dev
```
Inicia o servidor local com hot reload.

---

###  Build
```bash
npm run build
```
Gera a versão otimizada para produção.

---

### Preview
```bash
npm run preview
```
Visualiza o build final localmente.

---
